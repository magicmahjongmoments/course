# Mahjong Course Materials


## Where the Files Are

The key files and folders in this repo are:

- **[index.qmd](https://github.com/magicmahjongmoments/course/blob/main/index.qmd)**  
  The main landing page or entry document for the course.

- **[posts/](https://github.com/magicmahjongmoments/course/tree/main/posts)**  
  A directory of `.qmd` content for individual lessons, modules, or posts.

Click the links above to view each file or folder directly on GitHub.

## How to Edit `.qmd` Files Using GitHub’s GUI

If you are editing content using GitHub’s web interface (no local setup required):

1. Navigate to the `.qmd` file you want to edit (for example, **index.qmd** or a file inside **posts/**).

2. Click the **pencil icon ("Edit this file")** in the upper-right corner.

3. Make your changes in the browser editor.  
   - Text sections use standard Markdown.  
   - Code blocks should be left intact unless you know they need.  changes.
   
4. Scroll down to **Commit changes**.
   - Write a short, descriptive commit message.  
   - Select **Create a new branch for this commit and start a pull request**.  
   
5. Click **Propose changes** and then open the pull request.
